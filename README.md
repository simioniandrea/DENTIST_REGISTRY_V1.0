# Dentist client and appointment registry


## Getting Started

- Insert name,surname, age and phone of client
- With add button in the client window you can book an appointment
- appointment format: 
  - date : gg/mm/aa 
  - hour : xx:xx- yy:yy (for example 9:00-10:00) 


# Installation
```bash
npm install
npm build
npm start
```
